About Jeopardy

Hello there!

This project was created because of the author's insistance that there aren't good pieces of software that help in editing and playing Jeopardy, either in person or with a shared screen (MS PPT is doable, but a hassle to set up and not free).

If you see something wrong, or want to request features, please submit your concerns here. If you are interested in the code, you will find it here.
